# SocialSiteForDeveloper

This single page application developed Using MERN(MongoDB,ExpressJS,React,NodeJs) stack. To visit the site [SocialSiteForDevlopers](https://socialsitefor-developers.herokuapp.com/)

### Prerequisites

after cloning the app install node dependencies for server

```
npm install
```
change the directory to the client

```bash
cd client
```
and install dependencies for client app


```
npm install
```

and run the app using the command

```
npm run dev
```
